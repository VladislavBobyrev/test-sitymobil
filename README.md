<h3  align="center"> Команды Git</h3> 


|                        |                     |
|------------------------|:--------------------|
| Установить зависимости | `npm i`             |
| Запустить локально     | `npm run serve`     |
| Собрать                | `npm run build`     |
| Проверить              | `npm run lint`      |
|                        |                     |
